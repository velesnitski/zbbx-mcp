# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone.

## Our Standards

- Be respectful and constructive
- Accept feedback gracefully
- Focus on what's best for the project

## Enforcement

Unacceptable behavior can be reported to the project maintainers. All reports will be reviewed.

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.
